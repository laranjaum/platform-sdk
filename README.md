# Platform-sdk
Frontend code for Trama Platform

# Developing

## Requirements

Install grunt-cli

	npm install -g grunt-cli
	
Install bower

    npm install -g bower

Install Yeoman

    npm install -g yo
    
Install Boilerplate

    npm install -g generator-cg-angular

Install this project's dependencies

	npm install
	bower install
 
## Running
 just run `grunt serve` task
